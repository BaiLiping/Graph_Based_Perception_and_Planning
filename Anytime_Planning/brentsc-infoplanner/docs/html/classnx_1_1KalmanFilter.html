Detailed Description

The KalmanFilter class contains static methods which execute the predict and update steps of a Kalman Filter, both with and without measurements.

The KFCovariance function takes as input a prior covariance matrix, and computes the Kalman filter prediction and update steps to calculate the posterior covariance matrix. Note that the covariance posterior is deterministic and does not depend on the measurement values.

Parameters
